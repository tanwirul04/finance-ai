# Warung Pintar AI — Sistem Multi-Agent Analisis Keuangan UMKM

Tugas UAS AI Applied — sistem analisis keuangan berbasis Gemini API.

## 1. Cara jalankan (5 menit)

1. Pastikan Node.js sudah terinstal (cek: `node -v`).
2. Buka folder ini di terminal, lalu jalankan:
   ```
   npm install
   ```
3. Dapatkan Gemini API key gratis di **https://aistudio.google.com/apikey** (tinggal login akun Google, klik "Create API key").
4. Buat `.env`, lalu isi:
   ```
   GEMINI_API_KEY=api_key_kamu
   PORT=3000

   ```
5. Jalankan servernya:
   ```
   npm start
   ```
6. Buka browser ke **http://localhost:3000**
